Student API Docs

/students

GET

Returns all students

Example response

[
  {
    id: 1,
    firstName: 'John',
    lastName: 'Doe',
    email: 'John.Doe@quinnipiac.edu'
  },
  {
    id: 2,
    firstName: 'Billy',
    lastName: 'Joe',
    email: 'Billy.Joe@quinnipiac.edu'
  }
]

POST

Creates a new student

Example request body

{ 
  firstName: 'John',
  lastName: 'Doe',
  email: 'John.Doe@quinnipiac.edu'
}

* firstName, lastName, and email fields are required

/students/<id>

'id' of student must be provided in route

Example: /students/1 would specify student with 'id' value 1

If no student with specified 'id' value exists, an error is returned

GET

Returns the specified student

Example response

{
  id: 1,
  firstName: 'John',
  lastName: 'Doe',
  email: 'John.Doe@quinnipiac.edu'
}

PUT

Updates the specified student

Example request body

{ 
  firstName: 'John',
  lastName: 'Doe',
  email: 'John.Doe@quinnipiac.edu'
}

* firstName, lastName, and email fields are required

DELETE

Deletes the specified student

Be careful -- student will be removed permanently!!!
